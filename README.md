# 🎓 College Club Website

A modern and responsive **College Club Website** built using **HTML, CSS, and Python (Streamlit)**.
This project is designed to showcase college club activities, events, gallery images, and student registrations in a clean and user-friendly way.

---

## 🚀 Features

* 🏠 Home page with attractive landing section
* ℹ️ About page with club details
* 🖼️ Gallery page for event photos
* 📝 Join page for student registrations
* 📱 Responsive design
* 🎨 Clean UI with CSS styling
* 🐍 Streamlit integration support

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **Python**
* **Streamlit**

## ▶️ How to Run Locally

### For HTML Website

1. Download or clone the repository
2. Open `index.html` in your browser

### For Streamlit App

Run the following command:

```bash
streamlit run streamlit_app.py
```
